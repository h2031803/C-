변수가 정의됐다 = 컴퓨터 RAM 메모리의 조각(일부)이 그 변수에 할당됨
가장 작은 메모리 단위 = 1바이트(8비트)
컴퓨터의 모든 데이터 = 비트로 표현(0과1)
변수에 값을 할당 했다 = 컴파일러와 CPU가 해당 값을 비트로 메모리에 저장

** void의 사용법 **
함수가 값을 반환하지 않을 때 => void로 함수를 정의
함수가 매개변수를 사용하지 않을 때 => 매개변수 대신에 void를 사용 (ex) int getValue(void))

sizeof(자료형or변수이름) => 해당 자료형의 메모리 크기를 바이트로 변환 -> 자료형과 메모리 크기의 매칭은 다른 아키텍쳐를 가진 컴퓨터 or 컴파일러에서 다르다
