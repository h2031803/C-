변수가 정의됐다 = 컴퓨터 RAM 메모리의 조각(일부)이 그 변수에 할당됨
가장 작은 메모리 단위 = 1바이트(8비트)
컴퓨터의 모든 데이터 = 비트로 표현(0과1)
변수에 값을 할당 했다 = 컴파일러와 CPU가 해당 값을 비트로 메모리에 저장

** void의 사용법 **
함수가 값을 반환하지 않을 때 => void로 함수를 정의
함수가 매개변수를 사용하지 않을 때 => 매개변수 대신에 void를 사용 (ex) int getValue(void))

sizeof(자료형or변수이름) => 해당 자료형의 메모리 크기를 바이트로 변환 -> 자료형과 메모리 크기의 매칭은 다른 아키텍쳐를 가진 컴퓨터 or 컴파일러에서 다르다

실수를 print했을때 보여주는 유효숫자는 6자리까지

#include <iomanip>
std::cout << std::setprecision(16) : 화면에 보여주는 유효숫자를 16으로 재설정하는 코드
float 값 자체가 저장할 수 있는 자료길이 = 유효숫자 6~9자리
double 값 자체가 저장할 수 있는 자료길이 = 유효숫자 15~18자리

const int = ~~  : 상수 변수 선언법 (초기화 까지 해주어야 함)

컴파일 : 코드가 문법적으로 정확한지 확인하고 코드를 object file로 바꿔줌
런타임 : 응용프로그램(exe)을 실행하는 시간(프로그램이 한 줄씩 실행)

런타임 상수 : 초깃값을 런타임에서만 확인할 수 있는 상수
컴파일 시간 상수 : 컴파일 과정에서부터 초깃값을 확인가능한 상수

constexpr int a = 9 : 컴파일 시간 상수를 선언할 때 사용

const 변수를 사용해서 매직넘버(의미를 알 수 없는 숫자)를 방지할 것 
(constexpr int maxStudentsPerClass { 30 }; ---> 심볼릭 상수라고 한다)
